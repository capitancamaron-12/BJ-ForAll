CHAPTER 02 — COUNTING SYSTEMS

Hi-Lo & Wong Halves Explained

Card counting assigns a numerical value to each card. As cards are dealt, you maintain a running total. When the count is high, the remaining deck favors the player — bet more.

01

True Count Formula

TC = RC ÷ Decks Remaining. Always convert running count to true count before making bet decisions. Round toward zero.

02

Ace Side Count

Track aces separately from the main count. When more aces remain than expected, increase your bet an extra unit.

03

Bet Spread

Professional minimum is 1:12. TC ≤ +1: min bet. TC +2: ×2. TC +3: ×4. TC +4: ×8. TC ≥ +5: maximum.

04

Shoe Penetration

Only worth counting when penetration exceeds 65–70%. Below that, the count never becomes significant enough to matter.
